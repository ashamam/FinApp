# FinApp
My pet project that will help people to track their finances effectivly.


Финансовый трекер

MVP
Цель:

Приложение предназначено для персонального учёта расходов и визуализации данных с акцентом на простоту, оффлайн-доступ и высокую скорость работы.

ЦА:
Пользователи, которые чувствуют перерасход; возраст 18–70. Фокус на простоту и оффлайн.

Карта пути:
окно регистрации
окно с текущим днем с кнопкой посередине добавить расход
при нажатии на кнопку происходит переход на окно добавление расхода с возможностью указать куда уходили финансы и с возможностью добавления комментария
окно с календарем(возможность менять представление между годом месяцем, днем)


Функциональные требования:

форма регистрации и возможность защитить данные паролем
возможность управлять календарем
редактирование движения финансов
круговая диаграма в режиме календаря день
рекомендации будут содержать в себе простые правила управления финансами взятые из интернета, в дальнейшем можно рассмотреть возможность подключения экспертов по подписке

Нефункциональные требования:
простой минималистичный дизайн, должен выглядеть современнее чем у конкурентов.
высокая скорость работы
возможность добавлять свои траты в офлайн режиме

Критерий приемки:
	Пользовательская история:
Как пользователь я хочу видеть суммарный расход при выборе: дня месяца или года.


	Критерий:
Приложение запускается менее чем за 2 секунды и корректно работает без подключения к интернету.
при выборе месяца пользователь должен видеть расход в правом углу экрана
при смене варианта отображения, должен меняться доход

	Пользовательская история:
Как пользователь я хочу видеть подробный анализ моих расходов по категориям
	
	Критерий:
При нажатии на кнопку расходы открывается круговая диаграмма. внизу списком по категориям сколько в валютной единицы было потрачено на категории
Под списком находится другой список содержащий уже конкретные траты, также разделенные на категории


Стек Технологий:
Jetpack Compose, Room (SQLite), Hilt (Dagger), MPAndroidChart, Material 3, SQLCipher for Room
